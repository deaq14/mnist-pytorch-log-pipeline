# mnist-pytorch-log-pipeline
Modular PyTorch implementation of an MNIST digit classifier with structured logging and loss visualization. Built in Jupyter for reproducibility and portfolio presentation. Includes training, evaluation, and performance tracking in a clean, extensible format.

## Getting Started

### Requirements

Install dependencies with:

bash

pip install torch torchvision matplotlib

Run the Notebook
Open Loggs.ipynb in Jupyter and run each cell sequentially. The notebook is organized into:
- Setup & Imports
- Data Loading
- Model Definition
- Training Loop
- Evaluation
- Loss Visualization
